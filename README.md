# Telegram Bot

A Telegram bot built with Node.js and Telegraf.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file from the example:
```bash
cp .env.example .env
```

3. Get your bot token from [@BotFather](https://t.me/BotFather) on Telegram and add it to `.env`:
```
BOT_TOKEN=your_actual_bot_token_here
```

4. Run the bot:
```bash
npm start
```

## Features

- Responds "Hi, what's your name?" to any text message
